## Installation

### Via VS Code Marketplace
1. Open VS Code
2. Go to Extensions (Ctrl+Shift+X)
3. Search for "Oceanic Park"
4. Click Install

### Via Command Palette
1. Open Command Palette (Ctrl+Shift+P)
2. Type: `ext install Parklife.oceanic-park`

1. Open Cursor
2. Go to Extensions
3. Search for "Oceanic Park" 
4. Click Install
